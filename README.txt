PASSO A PASSO PARA DEPLOY:

1. Acesse https://vercel.com e clique em "Add New Project"
2. Importe esse projeto via GitHub ou upload manual
3. Configure como Next.js
4. Adicione as variáveis do arquivo `.env.local`
5. Clique em Deploy

O painel do criador estará disponível em /
Os apps gerados estarão em /app/[slug]
